ST7789 basic project
====================

This project can be used to test:

1. ST7789 display modules
2. XPT2046 touch controllers

for basic connections.

## How to build?

- Use vscode with RaspberryPi PICO extension
- Ensure the pico sdk is installed
- Use vscode tasks to Compile -> Flash board

## Output

- Use a uart-usb converter to read uart traffic into a terminal
